# Multimedia Blog

This is a multimedia blog webpage created using basic HTML. It showcases audio, images, and videos in a structured format.

## Features
- Audio playback using `<audio>`
- Video playback using `<video>`
- Image display using `<img>`
- Simple and clean HTML structure

## Description
The project demonstrates HTML tags like `<audio>`, `<video>`, `<img>`, headings, and text formatting. It helps in understanding how to embed and display multimedia content on a webpage.
